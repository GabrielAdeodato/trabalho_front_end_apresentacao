Aprenda o passo a passo de:

Como adcionar arquivos via terminal git pela primeira vez em 3 passos.

1- Primeiro criamos repositório pasta local

2- depois criaremos um arquivo como por exemplo um arquivo README.md Escreva: echo "# teste" >> README.md

3- então iniciaremos o git no repositório local , caso ainda não tenha sido iniciada
3.1- Escreva: git init

4- Depois enviamos um comando para o git adicionar um arquivo para ser comitado é enviado pro github
4.1- git add README.md
4.2- Caso deseje enviar varios arquivos basta escrever “git add . ”

5- Depois daremos um commit nós ou no arquivo adicionado . Um commit não passa de um texto indicando as alterações que foram feitas.
5.1- Escreva: git commit -m "commit exemplo"

6- Agora indicaremos ao computador em qual caminho/branch de desenvolvimentos gostaríamos de adicionar os arquivos , como estamos na principal , enviaremos como exemplo pra main .
6.1- Escreva: git branch -M main

7- Agora indicamos o repositorio on-line que queremos enviar remotamente o arquivo com o commite
7.1- Escreva: git remote add origin <"url do repositorio on-line"> (tire as aspas)

8- É pronto basta enviar com o seguinte comando
8.1- Escreva: git push -u origin main

Obs: Caso nunca tenha enviado ou logodao seu git no computador em questão sera pedido quer apos o passo anterio vc logue uma vez em sua conta Github , refazer o comando a partir do tópico 3.6

Como adicionar arquivos via terminal git em 3 passos.

Antes de tudo, abra o terminal

1- Escreva no terminal (.git remote add origin <"url do repositorio remoto">) (tire as aspas)

2- Escreva no terminal (.git branch -M main)

3- Escreva no terminal (.git push -u origin main)
