O que é GitHub?

GitHub é uma espécie de rede social para programadores onde o usuário pode criar repositórios para guardar seus projetos em nuvem, porém não é só isso, o GitHub serve também para compartilhar seus códigos com o resto do mundo com projetos open source.

Além de outras funcionalidades, o GitHub tem como tema principal o controle de versões do software que o usuário esta produzindo, tanto sozinho quanto em parceria com outras pessoas.

O Git é uma ferramente extremamente importante para quem está desenvolvendo projetos que precisarão de atualizações durante o tempo, pois com ele o usuário é capaz de controlar cada versão do código.

A ideia do git é que o usuário possua o controle do que foi feito em cada alteração para caso exista a necessidade de recuperar esse código ele consiga recuperar com facilidade.
